## 📌 项目简介

本发明公开了一种面向任务可靠性约束的无人机集群多批次动态轮换调度方法。通过**实时OODA环状态感知**、**混合事件驱动状态转移模型**与**三级优先级轮换派遣策略**，在有限资源下最大化无人机集群的任务可靠性，显著提升长时间、高强度任务的持续执行能力。

> **适用场景**：持续监视、协同打击、应急救援、边界巡逻等需要高可靠性的无人机集群任务。

---

## 🧩 背景与挑战

无人机集群在长时间任务中面临以下核心挑战：

| 挑战 | 具体表现 |
|------|----------|
| ❌ 缺乏系统性分批理论 | 依赖经验性分批或简单规则（如按功能均分），缺乏考虑任务剖面、故障概率、燃油状态等因素的动态分批方法 |
| ❌ 批次间协同机制薄弱 | 时序轮换衔接、空间覆盖互补、功能配合支援等复杂时空耦合关系建模不足 |
| ❌ 与任务可靠性评估脱节 | 调度方法未能将任务可靠性的动态预测深度融入决策循环，缺乏实时指标作为优化目标 |
| ❌ 计算复杂性与实时性矛盾 | 多约束、多目标、高不确定性调度问题计算复杂度高，难以在线求解 |

本发明通过**量化评估 + 动态调度**系统性地解决了上述问题，为无人机集群任务规划提供了科学的定量化决策支持。
