## 📌 摘要

本发明公开了一种面向任务可靠性约束的无人机集群多批次动态轮换调度方法。通过解析输入任务参数，生成可形成初始OODA环的可行体系组合；针对每个组合建立基于连续时间马尔可夫链的单机状态转移模型，制定当某型无人机失效时立即从基地派遣同型替补的动态轮换策略；构建以OODA环闭合时间占比不低于设定阈值为准则的任务成功判定体系，建立以任务可靠性最大化为目标、平均正常工作时间占比为辅助目标的多目标优化模型；采用蒙特卡洛仿真模拟随机故障与轮换过程，统计各组合的任务可靠性，输出最优目标地部署与基地候补组合。本发明实现了在资源约束下无人机集群任务可靠性的量化评估与动态调度优化，显著提升了持续能力与资源利用效率。

> **适用场景**：持续监视、协同打击、应急救援、边境巡逻等需要高可靠性的无人机集群任务。

## 背景与挑战

无人机集群分批次调度是应对系统退化、故障频发及动态环境挑战的关键技术。然而，现有研究存在明显不足：

| 问题 | 描述 |
|------|------|
| ❌ 缺乏系统性分批理论 | 依赖经验性分批或简单规则，未考虑任务剖面、故障概率、燃料状态等 |
| ❌ 批次间协同机制薄弱 | 对时序轮换衔接、空间覆盖互补等复杂耦合关系建模不足 |
| ❌ 与任务可靠性评估脱节 | 调度方法未能将可靠性动态预测深度融入决策循环 |
| ❌ 计算复杂性与实时性矛盾 | 多约束、多目标、高不确定性调度问题计算复杂度高 |

本发明通过**量化评估 + 动态调度**系统性地解决了上述问题。
